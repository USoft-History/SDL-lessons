# Lesson 1 — Application Loop Fundamentals

В этом уроке мы создадим первое полноценное SDL2-приложение и разберём основные элементы, которые используются практически в каждой игре.

После завершения урока вы будете понимать, как создаётся окно приложения, как работает главный игровой цикл и каким образом происходит отрисовка кадров на экране.

## Что изучим

В рамках урока рассмотрим следующие функции и концепции:

* SDL_Init
* SDL_CreateWindow
* SDL_CreateRenderer
* Главный цикл приложения
* SDL_PollEvent
* Очистка экрана
* SDL_RenderPresent
* Корректное завершение приложения

---

## Структура SDL2-приложения

Практически любая SDL2-программа состоит из нескольких этапов:

1. Инициализация SDL.
2. Создание окна.
3. Создание рендера.
4. Запуск главного цикла.
5. Обработка событий.
6. Отрисовка кадра.
7. Завершение работы приложения.

## Разбор кода

**SDL_Init**

SDL_Init(SDL_INIT_VIDEO);

Инициализирует видеоподсистему SDL.

Без вызова этой функции работа с окном и рендерингом невозможна.
