# SDL2 for PlayStation 2 and PC Homebrew Lessons

Серия практических уроков по разработке кроссплатформенных приложений и игр на SDL2 для Linux, Windows и PlayStation 2.

В репозитории рассматриваются:

* установка и настройка SDL2 под Linux;
* установка официального тулчейна PS2DEV;
* подготовка окружения для разработки под PlayStation 2;
* создание и сборка первых SDL2-приложений;
* работа с графикой, вводом и звуком;
* создание кроссплатформенных проектов с единым исходным кодом для ПК и PS2.

Курс ориентирован на начинающих разработчиков, которые хотят изучить SDL2 и познакомиться с разработкой Homebrew-приложений для PlayStation 2.
