# Lesson 0.1 - Project Setup

## Цель урока

Подготовить проект для разработки на SDL2 под Ubuntu.

После выполнения урока у вас будет:

* установлен компилятор GCC;
* установлен SDL2;
* создана структура проекта;
* создан первый исходный файл;
* готовность к написанию SDL2-приложений.

## Создание проекта

После того как мы развернули SDL2 под нужную платформу, можно создать 

Создадим каталог проекта:

```bash
mkdir sdl2-tutorial
cd sdl2-tutorial
```

Создадим структуру каталогов:

```bash
mkdir src
mkdir bin
```

Структура проекта должна выглядеть так:

```text
sdl2-tutorial/
├── bin/
└── src/
```

---

## Создание первого исходного файла

Создайте файл:

```text
src/main.c
```

Содержимое файла:

```c
int main()
{
    return 0;
}
```

Теперь структура проекта выглядит так:

```text
sdl2-tutorial/
├── bin/
└── src/
    └── main.c
```

Если все установилось правильно можете из **sdl2-tutorial/** запустить терминал и выполнить:

PS2:
```bash
/usr/local/ps2dev/ee/bin/mips64r5900el-ps2-elf-gcc src/main.c -o bin/lesson_0_1.elf
```

Linux:
```bash
gcc src/main.c -o bin/lesson_0_1.app
```
