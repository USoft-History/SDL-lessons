# Инструкция по развертыванию SDL2 окружения для linux

## Установка необходимых пакетов

Откройте терминал и выполните:

```bash
sudo apt update

sudo apt install build-essential
sudo apt install libsdl2-dev
```

### Что устанавливается

**build-essential**

Содержит основные инструменты разработки:

* GCC
* G++
* Make
* стандартные библиотеки разработки

**libsdl2-dev**

Библиотека SDL2 и заголовочные файлы для разработки.

---

## Проверка установки GCC

Проверьте, что компилятор установлен:

```bash
gcc --version
```

Пример вывода:

```text
gcc (Ubuntu 13.x.x) ...
```

---

## Проверка установки SDL2

Выполните:

```bash
sdl2-config --version
```

Пример вывода:

```text
2.30.x
```

Если версия отображается, SDL2 установлена корректно.
