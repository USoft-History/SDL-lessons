# # Lesson 4.3 - Multiple Text Labels

В этом уроке мы научимся отображать несколько текстовых надписей одновременно с помощью библиотеки SDL_ttf.

До этого мы выводили только одну строку текста. В реальных приложениях и играх обычно требуется отображать сразу несколько текстовых элементов: название игры, счёт игрока, количество жизней, FPS, подсказки управления и другую информацию интерфейса.

---

## Что изучим

* Использование нескольких текстовых меток на одном экране
* Создание нескольких `SDL_Texture` из текста
* Размещение текста в разных частях экрана
* Использование различных цветов текста
* Использование нескольких размеров шрифта
* Освобождение ресурсов SDL_ttf

---

## Используемые функции

### Инициализация SDL_ttf

```c
TTF_Init();
```

Инициализирует библиотеку SDL_ttf.

---

### Загрузка шрифта

```c
TTF_OpenFont("assets/font.ttf", 24);
```

Загружает шрифт указанного размера.

---

### Создание поверхности с текстом

```c
TTF_RenderUTF8_Blended(font, "Hello", color);
```

Создаёт поверхность (`SDL_Surface`) с отрендеренным текстом.

---

### Создание текстуры

```c
SDL_CreateTextureFromSurface(renderer, surface);
```

Преобразует поверхность в текстуру, которую можно вывести на экран.

---

### Отрисовка текста

```c
SDL_RenderCopy(renderer, texture, NULL, &rect);
```

Отрисовывает текстуру в указанной области экрана.

---

## Цвета текста

В уроке используются несколько цветов:

```c
SDL_Color white  = {255, 255, 255, 255};
SDL_Color yellow = {255, 255,   0, 255};
SDL_Color green  = {  0, 255,   0, 255};
SDL_Color cyan   = {  0, 255, 255, 255};
```

---

## Создание нескольких шрифтов

В примере используются два размера шрифта:

```c
TTF_Font *mediumFont = TTF_OpenFont("assets/font.ttf", 24);
TTF_Font *smallFont  = TTF_OpenFont("assets/font.ttf", 18);
```

Это позволяет выделять заголовки и обычный текст.

---

## Создание текстовых меток

В уроке создаются четыре независимые текстовые надписи:

### Заголовок

```text
SDL2 Text Demo
```

### FPS

```text
FPS: 60
```

### Очки

```text
Score: 100
```

### Подсказка

```text
Press ESC to quit
```

Каждая надпись имеет:

* собственную поверхность (`SDL_Surface`)
* собственную текстуру (`SDL_Texture`)
* собственную позицию (`SDL_Rect`)
* собственный цвет

---

## Результат

После запуска программы на экране отображается интерфейс из нескольких текстовых элементов:

```text
                SDL2 Text Demo

FPS: 60

Score: 100

Press ESC to quit
```

---

## Очистка ресурсов

После завершения программы необходимо освободить все текстуры:

```c
SDL_DestroyTexture(textureTitle);
SDL_DestroyTexture(textureFPS);
SDL_DestroyTexture(textureScore);
SDL_DestroyTexture(textureHint);
```

Закрыть шрифты:

```c
TTF_CloseFont(mediumFont);
TTF_CloseFont(smallFont);
```

Завершить работу SDL_ttf:

```c
TTF_Quit();
```
