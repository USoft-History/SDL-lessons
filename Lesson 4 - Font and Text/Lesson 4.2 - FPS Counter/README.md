# Lesson 4.2 - FPS Counter

В этом уроке мы научимся создавать **динамический текст** с помощью SDL_ttf и реализуем простой счётчик FPS (Frames Per Second).

В предыдущем уроке текст создавался один раз и никогда не изменялся. Теперь мы будем обновлять текст во время работы программы и отображать текущую производительность приложения.

---

# Что изучим

В этом уроке рассмотрим:

* SDL_GetTicks()
* Подсчёт кадров
* FPS (Frames Per Second)
* Формирование строк через `snprintf()`
* Обновление текста во время выполнения программы
* Пересоздание текстур при изменении текста
* Освобождение ресурсов

---

# Что такое FPS

FPS (Frames Per Second) показывает количество кадров, отображаемых за одну секунду.

Например:

```text
FPS: 60
```

означает, что приложение отображает примерно 60 кадров в секунду.

Чем выше FPS, тем плавнее выглядит изображение.

---

# Подготовка переменных

Создаём переменные для подсчёта кадров:

```c
int frameCount = 0;
int fps = 0;
```

`frameCount` хранит количество отрисованных кадров за текущую секунду.

`fps` содержит итоговое значение FPS.

---

# Получение времени

Для измерения времени используется функция:

```c
SDL_GetTicks()
```

Она возвращает количество миллисекунд с момента запуска SDL.

Создаём переменную:

```c
Uint32 lastFPSUpdate = SDL_GetTicks();
```

Она будет хранить момент последнего обновления FPS.

---

# Создание стартового текста

Создаём текст:

```c
char fpsText[64];

snprintf(
    fpsText,
    sizeof(fpsText),
    "FPS: %d",
    fps);
```

После запуска программа покажет:

```text
FPS: 0
```

---

# Создание поверхности

Создаём поверхность с текстом:

```c
SDL_Surface* fpsSurface =
    TTF_RenderUTF8_Blended(
        mediumFont,
        fpsText,
        white);
```

Результатом будет изображение текста в памяти.

---

# Создание текстуры

Преобразуем поверхность в текстуру:

```c
SDL_Texture* fpsTexture =
    SDL_CreateTextureFromSurface(
        renderer,
        fpsSurface);
```

После этого текст можно отображать через SDL_Renderer.

---

# Центрирование текста

Размер текста автоматически определяется библиотекой SDL_ttf:

```c
fpsSurface->w
fpsSurface->h
```

Используем эти значения для центрирования:

```c
SDL_Rect fpsRect =
{
    (SCREEN_WIDTH - fpsSurface->w) / 2,
    (SCREEN_HEIGHT - fpsSurface->h) / 2,
    fpsSurface->w,
    fpsSurface->h
};
```

---

# Подсчёт кадров

Каждый кадр увеличиваем счётчик:

```c
frameCount++;
```

Это позволяет узнать, сколько кадров было отрисовано за последнюю секунду.

---

# Проверка времени

Получаем текущее время:

```c
Uint32 currentTime =
    SDL_GetTicks();
```

Проверяем, прошла ли одна секунда:

```c
if (currentTime - lastFPSUpdate >= 1000)
{
    ...
}
```

1000 миллисекунд соответствует одной секунде.

---

# Вычисление FPS

Когда проходит секунда:

```c
fps = frameCount;

frameCount = 0;

lastFPSUpdate = currentTime;
```

Текущее количество кадров становится значением FPS.

После этого счётчик кадров обнуляется и начинается новый цикл измерения.

---

# Обновление текста

Формируем новую строку:

```c
snprintf(
    fpsText,
    sizeof(fpsText),
    "FPS: %d",
    fps);
```

Например:

```text
FPS: 60
```

или

```text
FPS: 144
```

---

# Создание новой поверхности

Поскольку текст изменился, необходимо создать новую поверхность:

```c
SDL_Surface* newSurface =
    TTF_RenderUTF8_Blended(
        mediumFont,
        fpsText,
        white);
```

---

# Проверка ошибок

Перед использованием поверхности желательно проверить результат:

```c
if (!newSurface)
{
    SDL_Log(
        "TTF_RenderUTF8_Blended error: %s",
        TTF_GetError());
}
```

---

# Обновление текстуры

Старая текстура больше не нужна:

```c
SDL_DestroyTexture(fpsTexture);
```

Создаём новую:

```c
fpsTexture =
    SDL_CreateTextureFromSurface(
        renderer,
        newSurface);
```

Теперь текстура содержит актуальное значение FPS.

---

# Обновление размеров текста

Размер строки может изменяться.

Например:

```text
FPS: 99
```

и

```text
FPS: 100
```

имеют разную ширину.

Поэтому обновляем размеры:

```c
fpsRect.w = newSurface->w;
fpsRect.h = newSurface->h;
```

и заново центрируем текст:

```c
fpsRect.x =
    (SCREEN_WIDTH - fpsRect.w) / 2;

fpsRect.y =
    (SCREEN_HEIGHT - fpsRect.h) / 2;
```

---

# Освобождение поверхности

После создания текстуры поверхность больше не нужна:

```c
SDL_FreeSurface(newSurface);
```

---

# Отрисовка FPS

Каждый кадр выводим текст:

```c
SDL_RenderCopy(
    renderer,
    fpsTexture,
    NULL,
    &fpsRect);
```

После этого показываем кадр:

```c
SDL_RenderPresent(renderer);
```

---

# Почему текст не обновляется каждый кадр

Обновление текста требует:

* создания новой поверхности;
* создания новой текстуры;
* выделения памяти.

Если выполнять это каждый кадр, производительность будет хуже.

Поэтому текст обновляется только один раз в секунду.

Это значительно эффективнее.

---

# Освобождение ресурсов

Перед завершением программы освобождаем созданные объекты.

Удаляем текстуру:

```c
SDL_DestroyTexture(fpsTexture);
```

Закрываем шрифт:

```c
TTF_CloseFont(mediumFont);
```

Завершаем SDL_ttf:

```c
TTF_Quit();
```

Завершаем SDL:

```c
SDL_Quit();
```
